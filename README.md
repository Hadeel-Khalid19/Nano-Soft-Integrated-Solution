# 🌟 Nano Soft Integrated Solution

![Project Status](https://img.shields.io/badge/Status-Complete-success)
![Version](https://img.shields.io/badge/Version-2.0.0-blue)

An enterprise-grade integrated solution developed for **Nano Soft**. This repository contains a high-performance Web Dashboard and a matching Mobile Application, both integrated with the latest Nano Soft Unified API (v2).

---

## 📂 Project Structure

| Module | Description | Tech Stack |
| :--- | :--- | :--- |
| 🌐 **nano_soft_web** | Premium Web Dashboard for user management and profile control. | React, Vite, Axios, Vanilla CSS |
| 📱 **nano_soft_mobile** | Professional Mobile Application for on-the-go account management. | Flutter, Dart, Clean Architecture |

---

## ✨ Key Features

### 🔐 Advanced Authentication Suite
- **Secure Login/Register:** Fully integrated with Nano Soft Auth v2.
- **Account Verification:** Multi-channel verification (SMS, WhatsApp, Email).
- **Security Recovery:** Robust "Forgot Password" and "Account Recovery" flows.

### 👤 Profile Management (The "impossible" made possible)
- **Dynamic Field Update:** Modify email and mobile with server-side validation.
- **Field Disabling:** Exclusive feature to securely clear or disable contact fields.
- **Real-time Sync:** Instant synchronization between web and mobile interfaces.

---

## 🛠️ How to Run

### 🌐 Web Dashboard
```bash
cd nano_soft_web
npm install
npm run dev
```

### 📱 Mobile Application
```bash
cd nano_soft_mobile
flutter pub get
flutter run
```

---

## 🚀 Deployment
The web solution is production-ready and can be deployed to any static hosting (Netlify, Vercel). The mobile solution is configured for Release APK building.
